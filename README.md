# darwin-lead-gen

Help me build an AI-powered BDR called Darwin.

You start by teaching Darwin about your ICP and then Darwin will find leads and generate personalized messages sent through linkedin and email in order to book meetings

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/darwin-lead-gen.git
cd darwin-lead-gen
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
